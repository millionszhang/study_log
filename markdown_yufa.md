markdown 文档的语法

- 标题

在 Markdown 中，如果一段文字被定义为标题，只要在这段文字前加`#`号即可。几级标题前面加几个#号，总共六级标题

![image-20210713153216871](C:\Users\zyw18\AppData\Roaming\Typora\typora-user-images\image-20210713153216871.png)



- 粗体与斜体

用两个`*`包含一段文本就是粗体的语法，用一个`*`包含一段文本就是斜体的语

![preview](https://pic3.zhimg.com/v2-632a68c6dc67118cce87314fca952936_r.jpg)

- 删除线

要加删除线的文字左右分别用两个~~号包起来。

![preview](https://pic4.zhimg.com/v2-3009ca2766f5e9ef05a787ca20627acb_r.jpg)

- 列表

在 Markdown 下，列表的显示只需要在文字前加上`-`, +或`*`即可变为无序列表，有序列表则直接在文字前加`1.2.3.`符号要和文字之间加上一个字符的空格。

![preview](https://pic3.zhimg.com/v2-45350a9a04e4a028989432814fed4d0a_r.jpg)

![preview](https://pic1.zhimg.com/v2-e3c91d6d28d79365b7abcb4848327e30_r.jpg)

- 饮用

只需要在文本前加入`>`这种尖括号（大于号）即可。引用也可以嵌套，如加两个>>三个>>>

![preview](https://pic2.zhimg.com/v2-2353becda84fa22d7b61ddcbf7dc7925_r.jpg)

- 列表嵌套

  上一级和下一级之音敲三个空格即可。

  ![preview](https://pic1.zhimg.com/v2-a4067ff3892e2fd1a96a0dacdaedda70_r.jpg)

- 图片与链接

  

插入链接与插入图片的语法很像，区别在一个`!`号

图片为：`![]()`

链接为：`[]()`

例如：![preview](https://pic4.zhimg.com/v2-3838b96ac32a6165e20f759b92a6c72b_r.jpg)



- 分割线

  三个或者三个以上的 - 或者 * 都可以。

![preview](https://pic4.zhimg.com/v2-09c78b5608c51e4b7c79472c06932983_r.jpg)



- 代码

  单行代码：代码之间分别用一个反引号包起来。

  ```text
  `代码内容`
  ```

  代码块：代码之间分别用三个反引号包起来，且两边的反引号单独占一行。

  ```text
  (```)
    代码...
    代码...
    代码...
  (```)
  ```
